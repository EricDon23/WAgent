# 项目重构文档

## 1. 重构概述

本次重构对AI对话生成模板系统进行了系统性优化，主要包括以下工作：

- **代码清理**：移除未使用的文件和代码块
- **文件合并**：整合功能相似的模块
- **交互功能实现**：开发promptAI交流系统
- **会话状态管理**：实现会话记忆功能
- **输出机制**：设计AI返回值的打印输出功能

## 2. 代码清理

### 移除的文件
- `src/core/prompt_ai.py` - 已被 `prompt_ai_enhanced.py` 替代
- `test_prompt_ai.py` - 测试已迁移到 `test_template_system.py`

### 清理的代码
- 移除了所有未被引用的导入和变量
- 删除了废弃的函数和方法
- 清理了无效的依赖项

## 3. 文件合并与模块整合

### 核心模块整合
- 将模板系统与PromptAI功能整合到 `src/core/prompt_ai_enhanced.py`
- 实现了统一的 `TemplatePromptAI` 类，包含所有核心功能

### 功能模块重组
- **会话管理**：集成到 `TemplatePromptAI` 类中
- **模板系统**：合并了模板加载、创建和管理功能
- **LangChain Prompt转换**：统一了Prompt构建和转换逻辑

## 4. 交互功能实现

### 新增功能
- **promptAI交流系统**：支持用户通过prompt与AI进行流畅交互
- **会话状态管理**：实现会话记忆功能，确保AI能够基于历史对话上下文进行内容延伸或修改
- **输出机制**：设计并实现了AI返回值的打印输出功能，确保交互结果清晰呈现

### 命令行接口增强
- 添加了会话管理命令：`session view`、`session clear`、`session status`
- 优化了 `_display_result` 方法，提供更清晰的输出格式
- 更新了所有 `prompt_ai` 引用为 `template_prompt_ai`

## 5. 会话状态管理

### 实现的功能
- **会话历史追踪**：记录用户和AI的对话历史
- **上下文感知**：基于历史对话生成上下文相关的prompt
- **会话管理**：提供会话历史查看、清空和状态查询功能
- **会话轮数计算**：准确跟踪对话轮数

### 核心方法
- `add_to_session_history()`：添加对话到历史记录
- `get_session_history()`：获取会话历史
- `clear_session_history()`：清空会话历史
- `get_session_rounds()`：获取会话轮数
- `_build_prompt_with_history()`：构建包含历史对话的prompt

## 6. 输出机制

### 增强功能
- **格式化输出**：提供清晰、结构化的AI响应输出
- **元数据显示**：展示token使用情况、生成时间等信息
- **用户友好提示**：添加操作提示和状态信息
- **错误处理**：优雅处理生成失败的情况

### 实现细节
- 优化了 `_display_result` 方法，支持多种输出格式
- 添加了详细的日志记录，便于调试和监控
- 实现了一致的错误处理和异常捕获

## 7. 系统架构

### 核心模块
1. **TemplatePromptAI** (`src/core/prompt_ai_enhanced.py`)：
   - 模板管理
   - 会话状态管理
   - AI文本生成
   - LangChain Prompt转换

2. **命令行接口** (`src/interaction/cli.py`)：
   - 用户交互
   - 命令处理
   - 结果显示

3. **MCP客户端** (`src/tools/mcp_client.py`)：
   - AI模型调用
   - 响应处理
   - 错误管理

### 数据流
1. 用户输入 → CLI处理 → TemplatePromptAI
2. TemplatePromptAI → 构建Prompt → MCP客户端
3. MCP客户端 → 调用AI模型 → 返回结果
4. TemplatePromptAI → 处理结果 → 更新会话历史
5. CLI → 显示结果 → 等待用户输入
